# 🎬 Amazon Prime TV Shows & Movies Analytics Dashboard

## 📌 Project Overview

This project is an interactive Exploratory Data Analysis (EDA) and Streamlit dashboard built using Amazon Prime Movies & TV Shows dataset.

The dashboard provides deep insights into:
- Content trends
- Ratings
- Popularity
- Runtime analysis
- Genre analysis
- Country-wise content production
- Movies vs TV Shows comparison

The project combines data analytics, visualization, and interactive UI to create a professional business intelligence dashboard.
